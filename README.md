# Lyrics

A music lyrics application for [Sailfish OS](https://sailfishos.org).

Currently it supports:
  * [ChartLyrics](http://chartlyrics.com)
  * [Genius](http://genius.com)
  * [LyricsMania](http://www.lyricsmania.com)
